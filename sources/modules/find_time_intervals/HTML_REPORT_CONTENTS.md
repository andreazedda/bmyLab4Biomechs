## üöÄ **ADVANCED REPETITION ANALYSIS** (Production-Ready)

**Only for original signals** (ANG1, ANG2, ANG3, Principal Component)

### Key Improvements Over Base Analysis

Visual highlight box explaining:
1. **Butterworth Filter:** Zero-phase low-pass (10Hz) preserves signal shape
2. **ROM Threshold:** Excludes half-reps and invalid movements
3. **Outlier Detection:** IQR method removes statistical anomalies
4. **First/Last Exclusion:** Removes adjustment and overshoot phases
5. **R¬≤ Validation:** Fatigue trend trusted only if R¬≤ > 0.5

### Advanced Quality Indices (0-100 Scale)

**Visual Cards with Scores:**

1. **ROM Consistency Index**
   - Formula: `100 √ó (1 - CV_range)`
   - **Interpretation:**
     - 80-100: Excellent consistency
     - 60-80: Good consistency
     - 40-60: Fair consistency
     - 0-40: Poor consistency (high variability)

2. **Tempo Consistency Index**
   - Formula: `100 √ó (1 - CV_duration)`
   - **Interpretation:** Rhythm regularity
   - High score = metronome-like execution

3. **Depth Index**
   - Formula: `100 √ó (ROM_mean / ROM_target)` or ROM consistency if no target
   - **Interpretation:** Achievement of target depth
   - Example: 90¬∞ squat target

### IGM Score (Integrated Global Metric)

**Large prominent card showing:**
- **Score:** 0-100 (weighted combination)
- **Formula:** `ROM√ó0.4 + Tempo√ó0.3 + Depth√ó0.3`
- **Qualitative Feedback:**
  - 80+: "Excellent: High consistency and depth control"
  - 60-80: "Good: Consistent performance with minor variations"
  - 40-60: "Fair: Noticeable variability, focus on consistency"
  - 0-40: "Poor: High variability, work on movement control"

**Research Value:**
- Single KPI for performance
- Comparable across sessions/athletes
- Weighted importance of different aspects
- Publication-ready metric

## üí° Tips for Researchers

1. **Compare Filtered vs Original:**
   - Validate filtering effectiveness
   - Ensure true signal not removed

2. **Cross-reference Analyses:**
   - Dominant frequency should match cycle detection
   - Autocorrelation period validates frequency domain
   - Phase statistics confirm cycle metrics

3. **Report Both Base and Advanced:**
   - Base for volume (all movements)
   - Advanced for quality (valid movements)
   - Ratio indicates technique consistency

4. **Use IGM Score for:**
   - Pre-post intervention comparisons
   - Athlete ranking
   - Proficiency classification

5. **Fatigue R¬≤ is Critical:**
   - Don't report slope alone
   - R¬≤ < 0.5 means high within-session variability
   - Significant p-value needs R¬≤ > 0.5 for interpretation
